# Story Documentation Template

## What This Does
Generates a complete, structured documentation page for any development story/task. Paste the template into any LLM along with your story details, and it produces a clean, professional document covering scope, steps, acceptance criteria, and testing.

## How To Use
1. Copy the contents of `story-doc.md`
2. Open any LLM (Claude, ChatGPT, Gemini, Copilot, etc.)
3. Paste the template
4. Replace the `{{PLACEHOLDERS}}` with your actual story details, OR just paste your raw story/ticket below the template and tell the LLM to fill it in
5. The LLM will output a formatted documentation page

## What You Get
- A professional document with consistent structure every time
- Covers: overview, pre-conditions, step-by-step implementation, scope boundaries, acceptance criteria, testing plan
- Works for any tech stack, any project, any team

## Tips
- You can paste a messy Jira ticket, Slack thread, or meeting notes — the template will force structure onto it
- If your story is light on details, the LLM will flag gaps for you to fill
- The output is ready to drop into Confluence, Notion, SharePoint, or any wiki
